<h1 align="center">
AirBnB clone - The console
</h1>

### Installation
- `$ *git clone "https://github.com/D4vc-198/holbertonschool-AirBnB_clone.git"*`
- `$ *cd holbertonschool-AirBnB_clone*`
- `$ *./console.py*`

### Execution
```
$ ./console.py
(hbnb) help

Documented commands (type help <topic>):
========================================
EOF  help  quit

(hbnb) 
(hbnb) 
(hbnb) quit
$
```

### Available commands:
- `all`
- `create`
- `count`
- `destroy`
- `help`
- `quit and EOF to exit the program`
- `show`
- `update`
